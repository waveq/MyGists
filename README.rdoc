== README
Aplikacja zaliczeniowa na przedmiot Architektura serwisów internetowych.

== Aplikacja w sieci
http://mygistsszymonrekawek.herokuapp.com/

== Opis
Zadaniem aplikacji jest przechowywanie kawałków kodu, języka w jakims kod został napisany oraz opisu.

== Funkcjonalności
* Kolorowanie kodu na stronie głównej oraz w showgists
* Podział na strony(na stornie mieści się 5 gistów)
* Ucinanie kodu na stronie głównej gdy jest zbyt długi (90 znaków)
* Wybór języka do naszego kodu jest bardzo prosty dzięki selectowi(nie musimy wpisywać nazwy języka, więc nie ma opcji pomyłki).
